# Content Editing Guide

The website is intentionally content-driven. Most future updates should happen in this folder.

## Files

- `profile.ts`: identity, affiliation, email, CV path, portrait path, navigation, social links.
- `research.ts`: modular research theme cards for the Research page.
- `publications.ts`: searchable publication records and topic filters.
- `projects.ts`: project and software cards.
- `news.ts`: homepage update strip.
- `media.ts`: gallery items and categories.
- `cv.ts`: CV page data.
- `notes/`: MDX placeholders for future research notes or blog-style updates.

## Rules

- Do not add fake publications, awards, or metrics.
- Mark placeholders clearly until real data is available.
- Use stable IDs for publications, projects, and media items.
- Keep image and video files in `public/assets` or `public/media`.
- Use external URLs only after they are ready to share publicly.
